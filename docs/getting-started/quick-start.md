# Quick Start

Fastest way to run the app locally.

## Prerequisites

- Docker
- Docker Compose
- OpenSSL

## Start the dev stack

```bash
git clone https://github.com/PhBassin/allo-scrapper.git
cd allo-scrapper

cp .env.example .env
openssl rand -base64 64
# paste the generated value into JWT_SECRET in .env

npm run dev
```

This starts:

- PostgreSQL on `5432`
- Redis on `6379`
- API server on `3000`
- Vite frontend on `5173`

Open `http://localhost:5173`.

## Enable scraping locally

The dev compose file does not start the scraper worker.

Run it separately:

```bash
cd scraper
RUN_MODE=consumer npm run dev
```

## Verify the backend

```bash
curl http://localhost:3000/api/health
```

Typical response:

```json
{
  "status": "healthy",
  "database": "connected",
  "timestamp": "2026-04-26T20:00:00.000Z",
  "cached": false
}
```

## First login

On a fresh database:

- username is `admin`
- password may be randomly generated by the migration runner and logged once on startup

Check the server logs for the generated admin password on first startup.
